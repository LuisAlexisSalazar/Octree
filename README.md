# Bienvenido al repositorio
**Octree**
Estructura de datos que cuenta con las clases Árbol, Nodo y Punto
## Instalación
```bash
cmake .
cmake --build .
```
## Necesario:
Para usar el ejemplo debemos copiar el archivo de source/rabbit.obj a la carpeta build/Debug/

## Ejecución
```bash
cd .\Debug\
.\octree.exe
```
